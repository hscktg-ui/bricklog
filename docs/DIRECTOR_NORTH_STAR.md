# BRICLOG Director's North Star

**갱신:** 2026-05-22 · **역할:** Brand Director — 제품·카피·품질·출시 기준의 단일 기준점

---

## 한 줄

**브랜드를 기억하는, 이번 주에 올릴 콘텐츠 운영 OS** — 주제 하나로 채널별 초안을 만들고, 검수 후 사용자가 직접 복사·업로드한다.

---

## 우리가 되는 것

| 축 | 정의 |
|----|------|
| **정체** | 로컬·소상공인 **Brand Content OS** (글 생성기·범용 챗봇 아님) |
| **결과** | 블로그 장문 + 스마트플레이스 공지 + 인스타 캡션(+ 이미지 프롬프트)이 **같은 브랜드 맥락**으로 나옴 |
| **차별** | 프로필·습관·피드백·성과가 **다음 글에 누적** — 매번 톤 설명·채널 재작성 최소화 |
| **신뢰** | 업로드 전 **길이·톤·placeholder·민감 표현** 게이트; 의료·법률 등은 보수적 가드레일 |
| **가격** | **3티어:** 무료(블로그 중심) → 브랜드 19,900원(3채널·메모리) → 스튜디오 39,000원(무제한 브랜드·고급 검수·주제 추천) |

---

## 우리가 하지 않는 것

- 네이버·인스타·플레이스 **직접 업로드·예약 발행** (복사 후 사용자 발행)
- API 키·비밀값 **클라이언트 노출**
- SEO 점수만 팔거나 무제한 저가량 생성기 포지션
- 모델명·스택 자랑형 개발자 도구

---

## 채널 원칙 (브랜드 디렉터 관점)

| 채널 | 목적 | 톤 | 금지 |
|------|------|-----|------|
| **블로그** | 이야기·신뢰·검색 의도 | 장문, 브랜드 화자 | placeholder, AI 관용구, 주제 이탈 |
| **플레이스** | 방문·공지 | 짧고 직접, 이모지 ≤2 | 블로그 문장 복붙, SEO·감성 수사 |
| **인스타** | 저장·공감 | 줄바꿈·감성, 이모지 밀도 규칙 | 블로그 톤, CTA 남발 |
| **이미지** | 썸네일·카드 프롬프트 | 블로그에서 파생 | 텍스트 오버레이 |

**피드백 루프:** 블로그 피드백 → 블로그 재생성 + (플랜 허용 시) 파생 채널 / 플레이스·인스타는 **채널 전용** 피드백·이모지 조절.

---

## 요금·권한 (코드 기준)

- **Free:** 월 5회, **블로그 중심** (`pipelineChannels: blog`만)
- **Brand:** 월 15회, blog+place+instagram, 브랜드 메모리, 민감 검수
- **Studio:** 월 30회, 전 채널+이미지, 고급 검수, 주제 추천

PG 결제·콘텐츠 캘린더는 **로드맵**; PricingModal·UsageMeter로 한도·업그레이드 안내.

---

## 품질·속도 기준

- **Core Quality Engine:** 90점 미만 재작성 (최대 5회, `contentOrchestrator`)
- **민감 업종:** compliance regen 1회 우선, 미통과 시 softPass + 전문가 확인 배너
- **로딩 UX:** 파이프라인 단계 라벨 명시; 불필요한 이중 regen은 attempt당 하나만 (compliance → core → constitution 순)

---
